Pourquoi TeamShuffle ?

Équilibrage intelligent

Algorithme qui analyse niveau et poste pour créer des équipes parfaitement équilibrées.

100% Offline

Fonctionne sans internet. Parfait pour les terrains où le réseau est limité.

Partage QR Code

Partagez votre profil via QR Code. Ajout instantané par scan.

Historique complet

Gardez trace de tous vos matchs et visualisez vos statistiques.

Avatars & numéros

Personnalisez votre profil avec avatars et numéros de maillot.

Mode Compétition

Organisez tournois et championnats avec gestion automatique.

Comment installer ?

1

Téléchargez le fichier APK en cliquant sur le bouton ci-dessus

2

Ouvrez le fichier téléchargé sur votre téléphone Android

3

Autorisez l'installation depuis "Sources inconnues" si demandé
